# Day 5 Lab
1. Pick a question you would normally answer by opening several documents.
2. Ask Glean Assistant that question in plain language, not as keywords.
3. Read the synthesized answer it returns.
4. Find the citations attached to the answer and note which sources it used.
5. Compare the time against doing the same lookup by hand.

## Working Example: Asking the Assistant

Glean Assistant answers a natural-language question by reading across your connected
apps and returning a synthesized response with citations. Verified against
`assistant_skills`.

```text
You:    What is our current PTO policy for new hires?
Glean:  New hires accrue 15 days of PTO in year one, prorated by start date. [1][2]
        [1] People Ops Handbook — "Time Off" (Confluence)
        [2] New Hire FAQ (Drive)

Ask a question when you want the answer assembled for you, not a list of links.
```
