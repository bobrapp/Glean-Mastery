# Day 6 Lab
1. Pick a query where you know which app holds the answer.
2. Add `app:` for that system (e.g. `app:slack`).
3. Compare results with and without the filter.
4. Try two apps to feel how it scopes.

## Working Example: app: in Action

The `app:` filter scopes a query to a single connected system. Verified against
`search_filters`.

```text
quarterly roadmap              → results from every connected app
app:gdrive quarterly roadmap   → only Google Drive
app:slack  quarterly roadmap   → only Slack messages
app:jira   quarterly roadmap   → only Jira issues

When you already know WHERE it lives, app: is the fastest cut you can make.
```
